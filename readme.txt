A simple application for sorting and selecting data according to predefined rules. 
The application works with lists of JSON objects of arbitrary structure, selects objects that contain keys with certain values, 
and sorts objects by values using the natural sort order.

Простое приложение для сортировки и отбора данных по заранее заданным правилам. 
Приложение работает со списками JSON объектов произвольной структуры, отбирает объекты, содержащие ключи с определенными значениями, 
а также сортирует объекты по значениям, используя стественный порядок сортировки.
